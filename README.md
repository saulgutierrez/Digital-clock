# Digital-clock
Simple digital clock made with JS. It has the function of show system time. With a classic Casio digital clock style you can see the real-time, second by second. <br>
Technologies used: <br>
<li> HTML </li>
<li> CSS </li>
<li> Vanilla JS </li>
